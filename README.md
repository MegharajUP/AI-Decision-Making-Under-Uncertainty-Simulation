# AI-Decision-Making-Under-Uncertainty-Simulation
AI Decision Making Under Uncertainty Simulation is a project that shows how AI makes smart decisions with incomplete information. It analyzes traffic conditions, sensor accuracy, and pedestrian requests, then uses probability-based reasoning to select the safest and most efficient action.

# Overview

This project demonstrates how an Artificial Intelligence (AI) system makes rational decisions when information is incomplete or uncertain. The simulation models a smart traffic signal control system that analyzes traffic conditions, sensor reliability, and pedestrian requests to select the most appropriate action.

# Features 

Simulates uncertain traffic conditions

Probability-based decision making

Traffic state classification (Low, Medium, High)

Intelligent traffic signal control

Simple and easy-to-understand implementation

Real-world AI concept demonstration

# Technologies Used

Python 3

Random Module

NumPy (Optional)

Matplotlib (Optional)

# System Workflow

Collect traffic data.

Introduce uncertainty through sensor confidence.

Analyze traffic conditions.

Estimate possible outcomes.

Select the best action.

Display the final decision.

# Decision Rules

High Traffic → Extend Green Signal

Medium Traffic → Keep Green Signal

Low Traffic + Pedestrians Waiting → Switch to Red Signal

Low Sensor Confidence → Choose Safe Action
